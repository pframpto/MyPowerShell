How to use this App

The first step is to choose from the list of configurations in the drop down box

Once you have a configuration selected. The key value pair that make up that configuration appear

Fill in the appropriate fields and for each one you want to include check the include checkbox. The optional ones are yellow.

There are hints provided for each field which you can read by clicking the ? button next to each field

Once you have the mandatory fields and the option fields you want to include, click on the "Copy Resouce button"

This copies a single individual resource to the clipboard which you can then copy to help make a configuration.

The skeleton for a DSC configuration

        configuration Example #
